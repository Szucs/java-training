Unit 15: More Maven

The goal of this unit is to introduce some of the basic Maven goals and to show how to define plug-ins and dependencies for useful facilities such as:

  • creating a jar file for your application;
  • creating a jar file for your source;
  • creating a jar file for javadoc;
  • running checkstyle analysis against your source code; and
  • running a code coverage utility against your source code.

You will also use the command-line utility, jar, to view the contents of the jar file that you create for your project. The jar utility is documented at http://docs.oracle.com/javase/6/docs/technotes/tools/windows/jar.html.

Your instructor will walk through the pom file that is supplied with the rpncalculator project on github.com/neopragma/java-training.
