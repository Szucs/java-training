Unit 6: Debrief

What did we learn in this unit? We learned:

  • The differences between the equality operator and the equals method.
  • How to ensure a custom class behaves properly when its equals method is called.
  • How to implement the Comparator interface in a custom class.
  • How to use basic Hamcrest matchers.
  • What happens when there is a version conflict between jar files on the classpath.
  • A little bit about how Java supports abstraction and inheritance.
